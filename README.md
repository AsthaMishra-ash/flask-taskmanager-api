# Flask Task Manager API

A simple REST API built with Flask to manage tasks.

## Features
- Add tasks
- View tasks
- Delete tasks

## Endpoints

GET /tasks  
Returns all tasks

POST /tasks  
Add a new task

Example request:
{
"title": "Finish project"
}

DELETE /tasks/<id>  
Delete a task

## Tech Stack
Python  
Flask  
SQLite  
SQLAlchemy

## Deployment
API deployed on Render
